# Data inventory

This directory contains compact derived datasets used in the CEUS manuscript.

## Included files

- `table1_population_building_weighted.csv`
- `gap_analysis_population_building_weighted.csv`
- `structural_operational_correlation.csv`
- `fire_station_accessibility_summary_gridbased.csv`
- `population_grid_250m.gpkg`
- `weighted_demand_points.gpkg`
- `priority_defense_corridors_rp100_strategic.gpkg`

## Notes

`priority_defense_corridors_rp100_strategic.gpkg` is a reduced publication-oriented subset containing RP100 strategic corridors only. The full workflow file `priority_defense_corridors.gpkg` is not included here because it exceeds standard GitHub file-size limits. For archival release, the full derived geospatial package should be deposited in a DOI-bearing repository such as Zenodo.
